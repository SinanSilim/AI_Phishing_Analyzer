# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-12

### Added
- Initial release of AI-Powered Phishing Analyzer
- URL analysis with domain age, SSL, WHOIS, and DNS checks
- Email content analysis with NLP and sentiment analysis
- OpenAI LLM integration for advanced linguistic analysis
- Have I Been Pwned API integration for breach checking
- Comprehensive risk scoring system
- Command-line interface (CLI)
- Graphical user interface (GUI) with Tkinter
- Batch analysis capability
- Configuration system with YAML
- Typosquatting detection
- Suspicious keyword detection
- Urgency and threat pattern detection
- URL extraction and analysis from emails
- Email address breach checking
- Export results to JSON
- Comprehensive documentation
- Example phishing and legitimate emails
- Unit tests
- Automated setup script

### Features
- Multi-factor risk assessment (0-100 scale)
- Four risk levels: Low, Medium, High, Critical
- Real-time analysis
- Configurable analysis components
- Support for both URLs and email text
- Detailed threat indicators
- Actionable recommendations
- Verbose and quiet output modes
- Progress indicators for long operations
- Color-coded risk display

### Security
- No data storage (privacy-focused)
- API key protection via gitignore
- Secure handling of sensitive information
- K-anonymity support for password checking
